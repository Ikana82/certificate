# FA02: Certification Hunter

## Ringkasan

Selain portofolio, sertifikasi bisa menjadi salah satu penilaian perusahaan untuk melihat kemampuan kandidat dalam pemahaman materi tertentu.

Pada tugas ini, kalian diminta untuk mendapatkan 4 sertifikat dengan harapan bisa menjadi awal habit baru untuk terus mencari sertifikasi lain kedepannya.

## Aturan & Kebijakan

- Waktu Pengerjaan: Week 3, Jumat, 15.00 WIB
- Student diharapkan menjunjung tinggi INTEGRITAS. Segala bentuk ketidakjujuran meliputi peniruan, plagiarisme, pemalsuan pengerjaan akan mendapatkan tindakan tegas dari akademik

## Github Live Code Workflow

Dalam pengerjaan ini, kalian diminta untuk melakukan commit sebagai checkpoin pengerjaan. Jika sudah mendapatkan sertifikat, segera lakukan `add-commit` dengan message relase yang jelas.

- Contoh 1: git commit -m "Sertifikat 1 Done"
- Contoh 2: git commit -m "Sertifikat 3-CSS: Done"
